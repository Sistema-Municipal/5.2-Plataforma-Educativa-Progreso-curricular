# Plataforma-Educativa---Progreso-Curricular-
Visualización en tiempo real del porcentaje de avance de un alumno dentro d euna carrera

 ## Funcionamiento

1. El alumno visualiza la lista de módulos.
2. Marca un módulo como completado.
3. El backend calcula el porcentaje de avance.
4. La barra de progreso se actualiza automáticamente.
